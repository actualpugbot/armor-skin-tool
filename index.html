Compatibility note

The verified downloadable armor-stand statue pack format is `128x80`, not `64x155`. This exporter matches that pack's `wood.png` and `woodBase` layout so the output is ready for the `woodNNN.png` workflow.

Pack reminder

`wood.png` replaces the default armor stand texture. `wood001.png` and other numbered files usually need matching `wood.properties` entries inside the resource pack.
